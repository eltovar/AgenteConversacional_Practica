REQUISITOS PARA ESTUDIO DE CRÉDITO – ARRIENDO

El proceso de estudio de crédito para arriendo con El Libertador es muy fácil:

REQUISITOS:
- Un Arrendatario + Un Deudor Solidario
- Cada uno debe contar con ingresos comprobables

BENEFICIOS DEL ESTUDIO:
- El estudio es 100% digital
- No tiene ningún costo (es gratis)

LINK PARA INICIAR EL PROCESO:
Para iniciar el estudio de crédito, ingresa al siguiente enlace:
https://analisisweb.ellibertador.co/estudio-digital/datos-basicos/natural

NOTA IMPORTANTE PARA EXTRANJEROS:
Para extranjeros el proceso de estudio se realiza por otro medio diferente al enlace digital.
Si eres extranjero, infórmale a tu asesor comercial para que te indique el proceso específico.

RESUMEN:
- Arrendatario + Deudor Solidario con ingresos
- Proceso 100% digital y gratuito
- Link: https://analisisweb.ellibertador.co/estudio-digital/datos-basicos/natural
- Extranjeros: proceso diferente, consultar con asesor
